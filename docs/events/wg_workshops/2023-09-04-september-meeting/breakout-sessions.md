# Breakout sessions

## Breakout session format

```{list-table}
:widths: 100 300
:header-rows: 0

- - 14:00 - 14:15
  - Vote for breakout 1 topics
- - 14:15 - 15:15
  - Breakout session 1
- - 15:15 - 15:30
  - Break, and vote for breakout 2 topics
- - 15:30 - 16:30
  - Breakout session 2
- - 16:30 - 16:45
  - Break
- - 16:45 - 17:15
  - Wrap up: Breakout groups present their next steps
```
